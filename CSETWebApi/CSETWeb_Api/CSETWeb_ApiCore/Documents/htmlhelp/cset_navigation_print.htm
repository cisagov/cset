CSET Navigation

 
When you select an assessment on the New Assessment tab the navigation panel is impacted.
 
Selecting a maturity Model will add specific information about that model (for example, tutorial or target selection), Maturity Questions to the Assessment tab, and, depending on the model selected, changes to the Results tab.
 
Selecting a standard adds Security Assurance Level, Standard Questions to the Assessment tab, and Analysis Dashboard, Control Priorities, Standards Summary, Ranked Categories, and Results by Category to the Results tab.
 
Selecting Network Diagram adds Security Assurance Level and Network Diagram to the Prepare tab, Diagram Component Questions to the Assessment tab, and Components Summary, Ranked Components by Category, Component Results by Category, Components by Component Type, and Network Warnings to the Results tab.
 
The following are shared among all features:
  • Assessment Configuration
  • Assessment Information
  • Executive Summary, Overview, and Comments
  • Reports
  • Feedback
