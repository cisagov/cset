Assessment Navigation

 
Upon selecting a new assessment or a previously created assessment, the assessment opens and displays the Prepare tab by default.
 

Assessment Tabs

  • The assessment tabs include the following options:
  • Prepare
  • Assessment
  • Results
  • User the assessment tabs or the assessment side navigation to navigate the assessment.
Top

Expand/Collapse

  • Select the Expand/Collapse button to show/hide the sidebar navigation.
Top

Home

  • Select the Home button to exit the assessment and return to the My Assessments tab.
Top

Assessment Side Navigation

  • The assessment side navigation includes the following headers:
  • Prepare
  • Assessment
  • Results
  • Sub-pages for each header display based on the selected standard or model, information entered into the Prepare section, and other settings.
  • Use the assessment side navigation or the assessment tabs to navigate the assessment.
