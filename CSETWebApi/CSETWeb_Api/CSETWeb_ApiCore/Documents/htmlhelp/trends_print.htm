Trends

 
Users can select the Trend option from the Tools dropdown menu to display the Trends window. Creating a trend allows users to visualize data collected across multiple assessments, review cumulative changes to an organization over time, and determine any trends.
 

New Trend

  • Select the New Trend button to create a trend and open the Trend Details window.

Trend List

  • The trend list displays the following information for any previously created trends:
  • Trend Name
  • The name of the trend entered on the Trend Details window.
  • Date
  • The date the trend was created.

Remove

  • Select the Remove button to delete the trend.
  • A Please Confirm dialog box displays the following message:
  • Select the Yes button to remove the trend.
  • Select the No button to keep the trend and close the dialog box.
