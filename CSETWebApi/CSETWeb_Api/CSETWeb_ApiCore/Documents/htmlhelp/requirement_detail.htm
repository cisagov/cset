Requirement Detail

 
Selecting the Create button on the Add Requirement window or selecting the Edit button on an existing requirement displays the Requirement Detail page. This page allows users to enter additional information about the requirement such as the Security Assurance Level (SAL), supplemental information, as well as attach reference documents and add related questions.
 

Requirement Detail

  • The Requirement Detail section includes the following fields:
  • Category
  • Subcategory
  • Identifier/Title
  • Standard Specific Requirement
  • The fields are auto-populated based on the information entered on the Add Requirement window.
Top

Security Assurance Level

  • Select the applicable Security Assurance Levels for the requirement.
  • See Security Assurance Level (SAL) for more information.
Top

Supplemental Information

  • Enter any supplemental information related to the requirement.
  • Use the buttons above the free text field to change the text's style, font, alignment as well as insert images and videos.
Top

References

  • Links to the Source Documents and any Additional Documents display under the References section for the requirement/question on the Assessment Screen.
  • Select the Manage Documents button to open the Standard Documents page.
  • Note: Before documents can be associated with the requirement, the user must associate the document with the module by checking the document's checkbox on the Standard Documents page.
  • The References section includes the following fields:
  • Source Documents
  • Select the source document from the dropdown menu.
  • This is the primary document that details the requirements for the standard.
  • Dropdown menu options are based on the selections made on the Standard Documents page.
  • Additional Documents
  • If applicable, select any additional documents from the dropdown menu.
  • This may include any secondary sources of information regarding the requirement that may be helpful to the assessor.
  • Dropdown menu options are based on the selections made on the Standard Documents page.
  • Bookmark
  • If the selected document contains pre-defined bookmarks, users can enter the  name of the desired bookmark.
  • Do not include a leading hash/pound symbol (#).
  • Note: Without an entered bookmark, the document will open on the first page. Documents with an entered bookmark, will open to the bookmark's corresponding page.
  • Select the respective Plus button to add and save the Source Document or Additional Document to the requirement.
  • Upon selection, another Source Document/Additional Document field displays.
Top

Related Questions

  • Associating questions with the requirement allows assessors to conducts assessments in either Questions Mode or Requirements Mode.
  • See Assessment Modes for more information.
  • Select the Add Question button to open the Add Question to Requirement page.
  • Once added, the questions will appear below the button.
  • Select the Change button to edit the question.
  • Note: Not all questions are editable.
  • Select the Remove button to delete the question from the requirement.
Top

Back to Requirement Listing

  • Select the Back to Requirement Listing button to return to the Requirement Listing page.
