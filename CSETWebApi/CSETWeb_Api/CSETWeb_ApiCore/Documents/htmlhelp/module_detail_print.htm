Module Detail

 
Users are directed to the Module Details page upon clicking the Create Module or Clone button on the Standard and Question Set Builder page. If creating a new module from a clone, the module details will be auto-populated based on the details from the source module.
 

Module Builder Navigation

  • Use the Module Builder Navigation links to access various pages throughout the Module Builder.
  • As the user progresses, additional page links will display.

Module Detail

  • The Module Detail section includes the following required fields:
  • Module Name
  • Cannot be an existing module name.
  • Short Name
  • Cannot be an existing short name.
  • Description
  • Enter a description of the module to display within the gallery on the New Assessment page and the Assessment Configuration page.

Requirements

  • Select the Requirements button to manage requirements and open the Requirement Listing page.
  • The button is enabled when all module details are valid.

Questions

  • Select the Questions button to manage questions and open the Question Listing page.
  • The button is enabled when all module details are valid.

Manage Documents

  • Select the Mange Documents button to manage documents and open the Standard Documents page.
  • The button is enabled when all module details are valid.

Clone From Existing Module(s)

  • Select the Clone from Existing Module(s) button to clone an existing module and open the Select Modules window.
  • The button is enabled when all module details are valid.

Back

  • Select the Back to Module List button to return to the Standard and Question Set Builder page.
