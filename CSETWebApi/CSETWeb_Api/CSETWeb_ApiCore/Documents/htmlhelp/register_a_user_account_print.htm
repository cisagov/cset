Register a User Account

 
Upon selecting Register New  Account from the Login page, users are directed to the Register a User Account page. To access CSET, users are required to have a registered account. CSET Desktop users will have an account made automatically.
 
 

Account Credentials

  • Enter the following required account credentials:
  • First Name
  • Last Name
  • Email
  • Confirm Email
  • Note: Users can not register an email address that is already associated with an existing account.

Security Questions

  • Security questions are optional, but provide an additional measure to ensure a user's identity when resetting a password.
  • Select an option from the Security Question dropdown field or create a custom security question.
  • Enter the answer in the Security Answer free text field.

Register

  • Select the Register button to register the user account.
  • Upon selection, the user will receive an email containing login instructions and a temporary password.

Login

  • Click Login to return to the Login page.

Forgot Password

  • If an existing user, click Forgot Password to reset the password.
  • Enter the email associated with the registered account.
  • Select the Next button to receive a temporary password and reset the previous password.
