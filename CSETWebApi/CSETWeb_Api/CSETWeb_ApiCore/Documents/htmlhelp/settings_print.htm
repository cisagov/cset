Settings

 
Users can select the Settings option from the User Profile dropdown menu to display the Settings window.
 

Password Encryption

  • Select the On button to require password encryption for assessment exports.
  • When enabled, the Encrypt Assessment window will display upon selecting the Export button.
  • Password encryption is turned off by default.
  • The toggle option highlighted in blue indicates the currently selected option.

User Language

  • Use the dropdown field to select one of the following available user languages. Translations are only available for the specified assessments.
  • English (Default)
  • Spanish
  • Automated Cybersecurity Evaluation Toolbox (ACET)
  • Ukrainian
  • Cybersecurity Framework 2.0 (CSF 2.0)
  • Translation is available only in Requirements Mode.
  • Cybersecurity Framework 1.1 (CSF 1.1)
  • Translation is available only in Requirements Mode.
  • Ransomware Readiness Assessment (RRA)
  • Cybersecurity Performance Goals (CPG)

Close

  • Select the Close button to save any changes and exit the Settings window.
