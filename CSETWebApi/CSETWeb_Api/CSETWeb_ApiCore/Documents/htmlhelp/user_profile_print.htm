User Profile

 
Users can select the User Profile option from the User Profile dropdown menu to display the User Profile window. This window allows users to update their name, email address, and security questions.
 

User Profile Fields

  • The User Profile window includes the following required fields:
  • First Name
  • Last Name
  • Email
  • Confirm Email
  • Fields are auto-populated with the credentials entered during the account registration process.

Security Questions

  • The User Profile window includes the option to add or update the user's security questions.
  • Security questions are optional, but provide an additional measure to ensure a user's identity when resetting a password.
  • Select an option from the Question dropdown field or create a custom security question.
  • Enter the answer in the Answer free text field.
  • If security questions were created during the account registration process, the fields will be auto-populated with the questions and answered previously provided.

Save

  • Select the OK button to save any changes and exit the User Profile window.

Cancel

  • Select the Cancel button to discard any changes and exit the User Profile window.
