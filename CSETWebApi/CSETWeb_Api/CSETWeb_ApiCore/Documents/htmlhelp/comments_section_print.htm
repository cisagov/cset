Comments Section

 
Selecting the Comments button on the assessment screen displays the Comments section.
 

Comments Section

  • Users can enter any comments pertaining to the question/requirement, such as explanations for the answer provided, implementation details, or reasons for marking a question for review.
  • A red dot displays next to the Comments button when a question/requirement has comments.
  • These questions/requirements are compiled into the Comments and Marked for Review report.
