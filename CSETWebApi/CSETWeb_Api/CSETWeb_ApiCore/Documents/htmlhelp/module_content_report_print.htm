Module Content Report

 
Users can select the Module Content Report option from the Tools dropdown menu to display the Module Content Report window. This window allows users to view the structure and content of a CSET module, such as an industry standard, maturity model, or best practices model.
 

Select a Module

  • Select a standard or model from the dropdown menu.

Launch Report

  • The Launch Report button is enabled when a standard or model has been selected.
  • Upon selecting the Launch Report button, the report opens in a separate tab.
  • Within the report, users have the ability to show/hide the guidance and references for all questions.
