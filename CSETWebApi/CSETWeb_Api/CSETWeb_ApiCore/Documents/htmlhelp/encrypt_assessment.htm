Encrypt Assessment

 
Upon selecting the Export button, if Password Encryption is enabled, the Encrypt Assessment window will display prior to downloading.
 

Encryption Password Fields

  • The Encryption Password section displays the following fields:
  • Password
  • Create a password for the encrypted assessment.
  • Users will be required to enter the password to access the encrypted assessment upon import.
  • Password Hint (Optional)
  • Create a password hint to display on the CSET File Import window.
Top

Cancel

  • Select the Cancel button to discard any changes and exit the Encrypt Assessment window.
Top

Continue

  • Select the Continue button to implement the assessment encryption password and export the assessment.
  • If a password is not entered in the encryption password field, the exported assessment will not be encrypted.
