Cloning and Building Modules in CSET
 
 
The Module Builder feature is located under the Tools menu at the top left-hand side of the CSET dashboard. The Module Builder allows a user to create a custom Standard or Question Set based on a pre-existing standard or assessment.
 
NOTE: "Module Builder" and "Import Modules" are only available on a local CSET install.

Create a New Module Feature
 
Figure: Create Module Button
 
Create Module Button 
 
Clicking the Create Module button will start the building of a new module and open up the Module Detail page. Clicking the Home link at the top left of the light-gray navigation bar will return you to the Module Builder home page and your list of any previously built or cloned modules.
 
 
 
 
Figure: Module Detail 
 
 
You can return to the Module List at any time by clicking the Home (described above) or the Back to Module List function at the bottom left of the Module Detail screen.
 
 
Module List (Standard and Question Set Builder page)
 
The Module List displays all custom modules created and edited by the CSET user.
 
 
Figure: Module List
    
 
 
To identify a cloned module in the Module List (Standard and Question Set Builder main page), the original module’s title is copied to the new clone, appending the word “(copy)” to the title. 
 
Clicking on the newly cloned filename will take you to the Module Detail page where you can fine-tune details such as Module Name, Short Name, Description, and industry Category.
 
Cloning a Module
 
There are two ways to clone a module in the Module Builder: the Clone button on the Module List (the Standard And Question Set Builder) page, and the Clone From Existing Module(s) button on the Module Detail page.
 
  Clone Module Button
 
 
To clone an existing module, begin by clicking the Clone Module Button for the module you wish to clone.  A copy of the module is created including requirements and questions which can be tailored to your specific industry or business function.
 
 
The second option for cloning an existing module can be found on the Module Detail page. 
 
To clone a module, click the Clone from Existing Module(s) button on the lower right of the Module Detail page.
 
 
 
 
 
 
 
 
Figure: Module Detail
 
After starting the cloning function, a popup menu allows you to select from a list of possible modules for cloning.
Figure: Select Module Screen
 
After choosing a module to clone (or more than one, if applicable) select the Add Questions and Requirements button in the popup. This will integrate your chosen assessment materials into your cloned assessment.
 
 
After cloning your chosen assessment material, you can further modify your cloned assessment.
 
From the Module Detail page and the Requirements, Questions, and Manage Documents buttons at the bottom right hand of the screen, you can begin editing and customizing your module using the Editing steps below.
 
 
Editing the content of a cloned module
 
After completing the cloning step and editing the Module Details for your cloned module, you can edit the content of the module including the Requirements Details, Security Assurance Levels, Supplemental Information, References, and Related Questions.

Deleting a Custom Module
 
  Delete Module Button 
 
A custom module may be deleted, provided no other modules have requirements based on it.  This restriction is enforced to maintain CSET assessment data integrity.
