Import New Module

 
Users can select the Import Modules option from the Tools dropdown menu to display the Import New Module window. The Import New Module window allows users to import new question or requirements sets into CSET.
 
Users can edit an existing standard, create their own module, or develop a schema in an external code editor and paste it into the module editor. The Import New Module window is designed for developer use and requires knowledge of JSON or XML.
 

Standards List

  • The Standards List displays a list of available standards with the following options:
  • XML
  • Select the XML button to export the standard as an XML file.
  • JSON
  • Select the JSON button to export the standard as a JSON file.
  • Load to editor
  • Select the Load to editor button to edit the standard in the Module Editor.
  • When a new standard is imported, the standard will appear in the Standards List and the assessment gallery on the New Assessment tab.
Top

Module Editor

  • Use the Module Editor to edit an existing standard or create a new standard for import.
  • Users can edit within the Module Editor or drag and drop a file into the editor.
  • Use CTRL+Spacebar to see list options while coding.
  • Use ALT+Shift+F to format code when loaded from the Standards List.
  • Note: New standards can contain both questions and requirements. If only using requirements, they will be duplicated for the questions set. Short Names must be unique when editing a previously used standard.
Top

JSON/XML Toggle

  • Use the JSON/XML toggle to select which language to use for the new imported standard.
  • Note: JSON is recommended as CSET has a more comprehensive validation and list options within the editor for the language.
Top

Schema

  • Use the Schema options to download an XML or JSON code schema to edit it in an external editor.
  • Upon completion in the external editor, drag and drop the file into the Module Editor.
Top

Expand

  • Select the Expand button to open the Module Editor in full-screen.
Top

Validation Messages

  • The Validation Messages box displays any errors within the Module Editor that conflict with the schema.
Top

Supporting Documents

  • Users can add supporting documents and references by dragging documents into the drop area.
Top

Import

  • Enabled when all errors within the Validation Messages box are resolved.
  • Select the Import button to import the standard into CSET.
