Observation Details

 
Selecting the Add an Observation button in the Observations section displays the Observation Details window.
 

Observation Details

  • The Observation Details section includes the following fields:
  • Title
  • Enter the title of the observation.
  • Importance
  • Assign an importance level of low, medium, or high to the observation.
  • Resolution Date
  • Enter a date by which the observation should be resolved.
  • Issue
  • Enter an explanation of the issue or problem which caused the question/requirement to be answered as "No."
  • Impacts
  • Enter the potential or real impacts caused by the issue on systems, assets, and/or procedures, etc.
  • Recommendations
  • Enter recommendations or steps for resolving the issue.
  • Vulnerabilities
  • Enter any known vulnerabilities related to the issue.

Individuals Responsible

  • The Individuals Responsible section includes a list of all contacts associated with the assessment.
  • Select one or more individuals to be responsible for resolving the issue identified in the observation.

Clear Selections

  • Select the Clear Selections link to uncheck all contacts within the Individuals Responsible section.

Contacts

  • The Contacts section displays a list of personnel associated with the assessment.
  • Contacts are added on the Assessment Information page or by selecting the Add Contacts button.

Add Contacts

  • Select the Add Contacts button to associate additional personnel with the assessment.
  • See Add Contacts on the Assessment Information page for more information.

Close

  • Select the Close button to save the observation and exit the Observation Details window.
