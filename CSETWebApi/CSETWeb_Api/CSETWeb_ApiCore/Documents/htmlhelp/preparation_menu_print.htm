Preparation Menu

 
The Preparation menu allows quick access to the assessment preparation screens. The figure below describes the buttons and menu.
 
NOTE: The Preparation menu will look different depending on what is selected on the Assessment Configuration screen. To learn more about navigation, see CSET Navigation.
 
Figure: Preparation button/menu
 

Prepare tab

 
Clicking the Preparation button will display the Assessment Configuration screen.
 
See Assessment Configuration for more information.

Navigation toggle

Use the Navigation Toggle to open and close the Navigation Menu.

Preparation menu

 
The Preparation menu items indicate the screens encountered by the user during the preparation process.
 
See Assessment Configuration, Security Assurance Levels (SAL), and  Cybersecurity Standards Selection for more information.
 
Note: The Cybersecurity Framework menu item isn't available unless Cybersecurity Framework is the selected standard.
