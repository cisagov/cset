Assessment Configuration

 
Selecting Assessment Configuration within the Prepare menu displays the Assessment Configuration page.
 

Organization Details

  • The Organization Details section includes the following fields:
  • Assessment Name
  • Enter the name of the assessment.
  • The assessment name will display on the My Assessments tab, the assessment's reports, and will be used as the default assessment file name upon export.
  • Displays "New Assessment" by default if a name is not entered.
  • Assessment Date
  • Enter the date of the assessment.
  • Displays the current date by default.
  • Selecting the field displays the date picker.
  • Facility Name
  • Enter the name of the facility where the assessment is to be performed.
  • City or Site Name
  • Enter the city or site name where the assessment is to be performed.
  • State/Province/Region
  • Enter the state, province, or region where the assessment is to be performed.
  • Note: Credit Union, Charter, and Asset fields will display when using the Automated Cybersecurity Examination Toolkit.

Assessment Type

  • The Assessment Type section displays a brief description of the standard or model.
