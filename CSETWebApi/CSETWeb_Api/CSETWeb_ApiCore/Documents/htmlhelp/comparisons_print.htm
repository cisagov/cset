Comparisons

 
Users can select the Compare option from the Tools dropdown menu to display the Comparisons window. Creating a comparison allows users to visualize data collected across multiple assessments.
 

New Comparison

  • Select the New Comparison button to create a comparison and open the Comparison Details window.

Comparison List

  • The comparison list displays the following information for any previously created comparisons:
  • Comparison Name
  • The name of the comparison entered on the Comparison Details window.
  • Date
  • The date the comparison was created.

Remove

  • Select the remove button to delete the comparison.
  • A Please Confirm dialog box displays the following message:
  • Select the Yes button to remove the comparison.
  • Select the No button to keep the comparison and close the dialog box.
