Requirement Listing

 
Selecting the Requirements button on the Module Detail page opens the Requirement Listing page where the user can add and manage the module requirements.
 

Module Builder Navigation

  • Use the Module Builder Navigation links to access various pages throughout the Module Builder.
  • As the user progresses, additional page links will display.
Top

Create Requirement

  • Select the Create Requirement button to create a new requirement and open the Add Requirement window.
Top

Requirement List

  • The Requirement List displays any previously created requirements or requirements cloned from a source module.
Top

Edit

  • Select the Edit button to edit the requirements and open the Requirement Detail page.
Top

Delete

  • Select the Delete button to remove the requirement.
  • A Please Confirm dialog box displays the following message:
  • Select the Yes button to remove the requirement.
  • Select the No button to keep the requirement and close the dialog box.
Top

Back

  • Select the Back to Module Detail button to return to the Module Detail page.
