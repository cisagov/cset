Share Data with DHS

 
After clicking Share Data with DHS from the Results navigation panel the Analytics screen will load.
 
Figure: CSET Share Assessment With DHS screen
 
The first tab displays "Pretty Data" that includes Question ID, Question Text, and Answers. Switching to Raw Data will show the same information in the format shown below.
 
Figure: Raw Data tab
 
At the bottom of the screen you have the option to Submit or Submit with Login.
 
Selecting Submit Anonymous will send your data (Question IDs, Questions, Answers) to be used in aggregation activities. Selecting Submit with Login will open up a login dialogue.
Note: If you haven't entered demographic information an error message will popup when trying to submit. Return to the Assessment Configuration screen to enter demographics.
 
Figure: CSET Analytics Login
 
Click Register User to create your CSET Analytics account.
 
Figure: Register for CSET Analytics account
 
Enter a username, email, and password (that follows the Password Rules below).
 
You will then be returned to the CSET Analytics login to enter you account information. After a successful login you will see the dashboard below.
 
Figure: CSET Analytics dashboard
 
The Analytic Assessments section will show all the assessments that you have submitted. Click on an assessment to see where you line up amongst your peers.
 
Figure: Assessment Comparison chart
 
The Assessment Comparison chart shows where your assessment data stacks up against both industry and sector averages. The sector and industry is based on what was provided in the demographics section of the Assessment Configuration screen.
 
Assessment: Displays the name of the assessment being compared as shown in the Analytic Assessments table.
 
Standard: Displays the standard that the assessment was ran against.
 
Sample Size: Displays the number of assessments being compared.
 
All Sectors: Select this button to see a comparison against all sectors not just the sector assigned to the assessment selected.
 
The chart displays the minimum, median, and maximum values of the cumulated assessment data.
