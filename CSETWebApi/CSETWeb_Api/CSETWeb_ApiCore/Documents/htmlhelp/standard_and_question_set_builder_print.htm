Standard and Question Set Builder

 
Users can select the Module Builder option from the Tools dropdown menu to display the Standard and Question Set Builder page. This page allows users to create custom modules.
 
Enterprise users will have access to custom modules create by others within the organization. However, CSET Desktop users will only be able to view custom modules created by themselves. To import a custom module, view Import New Module for more information.
 

Create Module

  • Select the Create Module button to create a new custom standard or question set and open the Module Detail page.

Module List

  • The Module List displays any previously created custom modules.
  • If available, select a module from the list to edit the module details.

Clone

  • Select the Clone button to open the Module Detail page and customize a cloned module that includes the same requirements and questions as the source module.

Delete

  • Select the Delete button to remove the custom module.
  • A Please Confirm dialog box displays the following message:
  • Select the Yes button to remove the module.
  • Select the No button to keep the module and close the dialog box.
  • Note: Modules that have been cloned to create new modules cannot be deleted.
  • Example: If a user clones an existing module, that module cannot be deleted as the new module's content is directly pulled from the source module.
