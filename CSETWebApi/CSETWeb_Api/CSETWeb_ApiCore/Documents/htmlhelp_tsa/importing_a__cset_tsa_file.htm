Importing a .cset tsa File

With the stand-alone version of CSET TSA running locally you can import a legacy CSET TSA file.
Much like importing a .cset tsa file click the button on the local Landing page.
The file explorer will open. Select the .cset tsa file to import. The upload dialogue will open. Once the dialogue has closed, refresh the landing page to see the imported assessment.
 
 
 
Exporting a CSET TSA Assessment
To export an assessment simply select the Export button next to the assessment to be exported on the Landing page.
 
Figure: Export button
 
After clicking the Export button the assessment will be downloaded as a .cset tsa file and will be in the user's Downloads folder (unless otherwise specified in browser settings).
 
 
 
 
CSET TSA Navigation
 
The features selected in the Assessment Configuration screen affect the CSET TSA navigation path.
 
Selecting Maturity Model adds Maturity Model Selection to the Prepare tab, Maturity Questions to the Assessment tab, and, depending on the model selected, changes to the Results tab.
 
Selecting a Standard adds Security Assurance Level and Cybersecurity Standards Selection to the Prepare tab, Standard Questions to the Assessment tab, and Analysis Dashboard, Control Priorities, Standards Summary, Ranked Categories, and Results by Category to the Results tab.
 
Selecting Network Diagram adds Security Assurance Level and Network Diagram to the Prepare tab, Diagram Component Questions to the Assessment tab, and Components Summary, Ranked Components by Category, Component Results by Category, Components by Component Type, and Network Warnings to the Results tab.
 
The following are shared among all features:
• Assessment Configuration
• Assessment Information
• Executive Summary, Overview, and Comments
• Reports
• Feedback
 
NOTE: Doing more than one feature type compounds the added sections. The navigation now stays static among all tabs.
