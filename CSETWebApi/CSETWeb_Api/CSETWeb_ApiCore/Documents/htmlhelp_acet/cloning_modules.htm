Cloning Modules

 
There are two ways to clone a module in the Module Builder: the Clone button on the Module List (the Standard And Question Set Builder) page, and the Clone From Existing Module(s) button on the Module Detail page.
 
To clone an existing module, begin by clicking the Clone Module Button for the module you wish to clone. A copy of the module is created including requirements and questions which can be tailored to your specific industry or business function.
 
 Figure: Clone button on the Module List
 
The second option for cloning an existing module can be found on the Module Detail page. 
 
To clone a module, click the Clone from Existing Module(s) button on the lower right of the Module Detail page.
 
Figure: Clone from Existing Module button
 
After starting the cloning function, a popup menu allows you to select from a list of possible modules for cloning.
 
Figure: Select Modules popup
 
After choosing a module to clone (or more than one, if applicable) select the Add Questions and Requirements button in the popup. This will integrate your chosen assessment materials into your cloned assessment.
 
After cloning your chosen assessment material, you can further modify your cloned assessment.
 
From the Module Detail page and the Requirements, Questions, and Manage Documents buttons at the bottom right hand of the screen, you can begin editing and customizing your module using the Editing steps below.
