Cyber-Informed Engineering (CIE) Analysis Configuration

Naming the Analysis

To realize the full benefits of this CIE Analysis tool, returning users can import previous answers as well as compare a previously completed Analysis as a baseline.

All CIE users must first name and save their Analysis accordingly. Naming the CIE implementation takes place in the CIE Analysis Configuration screen. Naming conventions should be considered when saving for ease of locating a particular Analysis in the future.

New CIE User

If you are a new user, the Import and Select Baseline fields below will be blank. Click “next” to begin your CIE Analysis.

Returning CIE User

A returning user will have a selection of previous CIE Analyses to return to on their dashboard.

After completing an initial CIE Analysis, the user will have the option to later recall the Analysis to:

  • Increase efficiency when prepopulating a future Analysis by importing answers from a previous CIE Analysis.

  • Track progress by choosing a previously completed CIE Analysis to serve as a baseline for comparison against the new Analysis.

    If you are a returning user, the fields below offer the ability to Import a previous Analysis and/or Select a Baseline Analysis from a previously saved CIE Analysis.

Import CIE Answers

Import all answers from a previous CIE Analysis. All answers, comments, and artifacts currently in the Analysis will be overwritten.
