Ransomware Readiness Tutorial

Accessing the Ransomware Readiness Assessment (RRA)

To use the RRA first follow the CSET installation instructions to properly install CSET.

Then do the following:

1. Login or start the CSET application

2. Start a new assessment

Figure: New Assessment

3. Select Security Model within the Assessment Configuration screen (this is the first screen you’re presented with after selecting “New Assessment”)

Figure: Select Security Model

4. Select Ransomware Readiness Assessment from the Security Model screen

Figure: Select Ransomware Readiness Assessment

5. Now you are set to complete the RRA assessment. Review the tutorial for additional instruction.
